ABSTRACT
UEORMS is a multi-tenant SaaS ERP platform that integrates inventory, finance, HR, CRM, purchase, projects, communication, and reporting operations in one system. It reduces fragmented tools and manual work by centralizing data and enforcing workflow approvals with role-based access control.
The project is built with React frontend, Node.js + Express backend, PostgreSQL database, and Prisma ORM. The codebase includes 19 backend modules, 124 frontend pages, and 125 relational models with tenant-aware architecture. The implementation demonstrates a practical enterprise-grade solution for scalable business operations.
